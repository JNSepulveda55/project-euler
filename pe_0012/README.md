# Highly Divisible Triangular Number
### Problem 0012

The sequence of triangle numbers is generated by adding the natural numbers. So the $7$<sup>th</sup> triangle number would be $1 + 2 + 3 + 4 + 5 + 6 + 7 = 28$. The first ten terms would be:
$$1, 3, 6, 10, 15, 21, 28, 36, 45, 55, \dots$$

Let us list the factors of the first seven triangle numbers:

$$\begin{align*}
\mathbf{1} &\colon 1\\
\mathbf{3} &\colon 1,3\\
\mathbf{6} &\colon 1,2,3,6\\
\mathbf{10} &\colon 1,2,5,10\\
\mathbf{15} &\colon 1,3,5,15\\
\mathbf{21} &\colon 1,3,7,21\\
\mathbf{28} &\colon 1,2,4,7,14,28
\end{align*}$$

We can see that $28$ is the first triangle number to have over five divisors.

What is the value of the first triangle number to have over five hundred divisors?

# Solutions

## Solution 1

To solve this problem we can use this nice trick for finding the number of divisors of a number. Let's see an example with the number $28$:

The prime factorizartion of $28$ is: $2^2 \cdot 7$

We can find the divisors of $28$ by trying out every combination of primes in its prime factorization:

- $1 = 1$
- $2^1 = 2$
- $2^2 = 4$
- $7 = 7$
- $2^1 \cdot 7 = 14$
- $2^2 \cdot 7 = 28$

We can see that 28 indeed has 6 divisors. However, the trick here is that we don't need to calculate each divisor, only count them. We can do this by just simple analysis:

$28$ is: $2^2 \cdot 7$, so we can use zero $2\text{s}$, one $2$ or two $2\text{s}$. Likewise, we can either use zero $7\text{s}$, one $7$, We have three options for $2$ and two options for $7$, then the number of divisors is $3 \times 2 = 6$, which is $(2 + 1)(1 + 1)$, where the first number in each parenthesis corresponds to the power of the prime in that position.

More generally, if we have a number $n = p_1^{e_1} \cdot p_2^{e_2} \cdot ...\,\cdot p_k^{e_k}$ for some $k$, then the number of divisors of $n$ or $\tau(n)$ is:

$$\tau(n) = (e_1 + 1)(e_2 + 1)...(e_k + 1) = \prod_{i=1}^{k} (e_i + 1)$$

Then, we just need to implement a function to find every $e_i$ and later use the formula to get the number of divisors of triangular numbers, returning the first one with over 500 divisors.

To find $e_i$ we also need to know $p_i$, so we might as well return them too. Also, the highest divisor of a number can't be greater than half the number, so we only need to check till $\frac{n}{2}$:

```py
def prime_decomposition(n: int) -> dict:
    """
    Find the prime decomposition of an integer.
    Return format is a dictionary with entries {prime: power}
    """

    if n <= 1:
        return {}  # Edge case
    
    decomposition = {}

    for i in range(n//2 + 1):

        if is_prime(i):
            if n % i == 0:
                decomposition[i] = 0  # Create dictionary entry for prime

            while n % i == 0:
                n //= i
                decomposition[i] += 1  # Update exponent for prime

        if n == 1:
            break

    return decomposition
```

We'll add this function to our [`utils.py`](../utils/utils.py) file.

Next, we need a function to calculate the number of divisors of a number from its prime decomposition:

```py
def number_of_divisors(n: int) -> int:
    """
    Calculate the number of divisors of n using its prime decomposition
    """
    decomposition = prime_decomposition(n)  # Decompose number in prime factors
    powers = decomposition.values()  # Extract the e_i from decomposition

    total = 1
    for e in powers:
        total *= (e + 1)

    return total
```

We'll add both of these functions fo our [utils](../utils/utils.py) file.

With this, we can find the number of divisors of any integer. Now we're ready to implement our final solution

```py
def solution_1(limit: int) -> int:
    n = 1
    triangular = 1

    while number_of_divisors(triangular) <= limit:
        n += 1
        triangular = n*(n+1)//2

    return triangular
```
